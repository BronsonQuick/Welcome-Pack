== Description ==

Welcome Pack is a BuddyPress plugin that enhances the new user experience. A newly-registered user is automatically made a member of a specified group and becomes friends with a specified member. A customisable "welcome" message can also be sent to the new user automatically.

== Installation ==

1. Upload `welcome-pack.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin.
1. Visit BuddyPress > Welcome Pack to configure the plugin.
