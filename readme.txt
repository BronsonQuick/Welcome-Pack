== Description ==

Welcome Pack is a BuddyPress plugin that enhances the new user experience. A newly-registered user is automatically made a member of a specified group and becomes friends with a specified member.

The next version of this plugin will allow for a customisable "welcome" message to be sent to the new user.

== Installation ==

1. Upload `welcome-pack.php` to the `/wp-content/mu-plugins/` directory
1. Visit Site Admin > Welcome Pack to configure the plugin.
